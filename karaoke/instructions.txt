If you ever want the VM path

  -SshPublicKey (Get-Content "$env:USERPROFILE\.ssh\id_ed25519.pub" -Raw)

vmreaper/init.py: returns immediately when the worker VM path is disabled.
infra/deploy.ps1: -DeployWorkerVm optional; validates SSH key only when that switch is used.

Same flow as the main page: upload one mixed audio file (vocals + band together). 
This page talks to a small Python server in this repo—not Azure Functions on your machine 
(func start is not used). 
Your file is saved into an input folder as the queue; a background thread runs Spleeter/Demucs 
and writes stems under an output folder; the player loads those files over HTTP.
Run:python karaoke/local_folder_queue.py (defaults: KARAOKE_LOCAL_ROOT = ~/.karaoke-local, port 8787, 
SEPARATOR=spleeter). Override base URL: ?api=http://127.0.0.1:8787 or set window.KARAOKE_API_BASE 
before scripts load. For the hosted cloud pipeline, use <a href="index.html">Karaoke Split (cloud).
