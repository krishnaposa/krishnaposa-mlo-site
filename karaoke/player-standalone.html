Offline karaoke player

Uses only karaoke-core.js — no Azure Functions, no API keys, no network for playback. Load stems from disk using the folder layout below (or pick two files).

Suggested folder layout

my-track/
  input/                 ← optional: original song (reference only here)
    song.mp3
  output/                ← stems to play (or use a folder named "stems")
    vocals.wav
    no_vocals.wav        ← or accompaniment.wav (Spleeter)

Use Choose project folder and select my-track — the player finds output/… or stems/… or Demucs folders. Or choose only the output folder.

Input folder (optional)

Reference only — lists audio-like files so you know what you uploaded. Not used for playback.

Stems (playback)

Pick a folder that contains vocals.* and no_vocals.* / accompaniment.* in the same directory, or choose two files.
